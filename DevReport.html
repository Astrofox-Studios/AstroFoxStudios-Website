DevReport

In-Depth look at Tester 4

TLDR;

If you aren't a dev or don't want to read all of this here is what happened: Destiny Games is run on a language called Skript, our Skript systems are normally quite efficient as seen in Wipeout at the end of the event, but in the cases of the lag seen on the streams we had a lot of inefficient systems.

VoidHunters

VoidHunters was one of the most if not the most laggy game of Tester 4 with it crashing the server, but luckily we now know why! VoidHunters was looping over 1600 blocks every time someone broke a concrete powder... I'm not kidding it was that bad. The main reason for it being coded this badly is that VoidHunters was coded nearly 2 years ago by Skye as one of her first games. You will all be happy to know this has already been fixed and we will be stress testing in the coming days! Also, we have added more of what we call 'Stop Checks'. These check to see if the /event PANIC command has been triggered and stop the game.

Cosmic

Cosmic's lag is a similar point, basically there are way too many things happening at once. For Cosmic, the case was way too many variables being set at the same time, because of this the server did something called hanging, basically it stops doing anything because it has too much to do. Sounds like me! This is being fixed by Squid right now but trust me it's not good at all as it is LOL. We are also going to add in a NEW x3 Loot pool... We have redone that about 10 times now.

Speed Through Space

The lag in Spped Through Space is basically the same. (See what I did there?) STS is built on the same framework as Cosmic, at checkpoints and starting the race, way too many variables are being set. Should be fixed by a simple timer and checkpoint rewrite though!

Over All,

The lag in the event is very fixable. Yes it will take us some time but as far as issues go this isn't the worst we have faced. (Yk the whole getting hacked mid event thing.) The event should resume pretty quickly and thanks for reading!

Also Wipeout Worked :D,

The only game that worked was itsasquid2's game "Wipeout Rush" so uh, good job itsasquid2!
